The Ultimate MIDI Player for Roblox "Visual Pianos"

The powerful desktop application for translating MIDI files into real-time keyboard presses in the Roblox game, "Visual Pianos".

Key Features

Advanced Piano Roll Visualizer

Watch notes scroll towards the keyboard in real-time, with a dynamic piano roll that highlights every key press and sustain.

Live MIDI Input

Connect your physical MIDI keyboard or controller to see your own playing visualized live within the application.

Playlist Management

Load multiple MIDI files, organize your playlist, and enjoy continuous playback with the auto-play next feature.

Musical Tools

Transpose MIDI playback to any key with a simple control, and adjust playback speed from 0.5x to 2.0x in real-time.

Custom Key Mapping

Open the intuitive key map editor to re-assign which keyboard key is triggered by any MIDI note.

In-Depth Song Statistics

Analyze any MIDI file with detailed stats, including note count, duration, note density, velocity range, and more.

How to Use

  1. Download and extract the `Advanced.MIDI.Keyboard.Simulator.zip` from the link above.
  2. Run the exe file. No installation is required.
  3. Click "Add Files" to load one or more .mid files into the playlist.
  4. Launch Roblox and join the game "Visual Pianos".
  5. In the simulator, select a song from the playlist and click "Play". A 3-second countdown will begin.
  6. Quickly switch to the Roblox game window to give it focus.
  7. Watch and listen as the application plays the song perfectly in-game!

About the Project

This application was designed specifically for players of the Roblox game "Visual Pianos", allowing musicians, learners, and hobbyists to visualize and perform MIDI compositions in-game without needing physical hardware.
