How to Use

  1. Download your preferred version (Classic or AI) from the buttons above.
  2. Extract the `.zip` file. Run the `.exe` inside (No installation required).
  3. (AI Version Only): Ensure you have Ollama installed if you want to use the AI features.
  4. Drag and drop `.mid` files directly into the application playlist.
  5. Go to the "AI Assistant" tab and click "‚ú® AI Auto-Configure" to optimize settings for the song.
  6. Click "Play". A countdown will begin.
  7. Quickly switch to the Roblox "Visual Pianos" window to give it focus.
  8. Enjoy the performance!

About the Project

This application was designed specifically for players of the Roblox game "Visual Pianos", allowing musicians, learners, and hobbyists to visualize and perform complex MIDI compositions in-game without needing physical hardware. The new v2.0 update introduces local AI to make configuring complex songs effortless.
